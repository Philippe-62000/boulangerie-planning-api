🚀 RÉCUPÉRATION RAPIDE (5 MINUTES) :
1. Installer Git : https://git-scm.com/download/win
2. Installer Node.js : https://nodejs.org/en/download/
3. Cloner : git clone https://github.com/Philippe-62000/boulangerie-planning-api.git
4. Backend : cd backend && npm install
5. Frontend : cd ../frontend && npm install
6. Configurer backend/.env avec les variables ci-dessus
7. Lancer : cd backend && npm start
